# Patterns

Each file in this directory documents a single control pattern for agentic systems.

Patterns are independent. Use them individually or in combination.

## Index

- [Approval Gates](./approval-gates.md) — require explicit approval before high-risk actions
- [Tool Allowlists](./tool-allowlists.md) — restrict which tools an agent can invoke
- [Blast Radius Containment](./blast-radius.md) — limit the scope of potential damage
- [Human-in-the-Loop](./human-in-the-loop.md) — insert human checkpoints into agent workflows
- [Logging and Audit](./logging-and-audit.md) — record agent actions for review and accountability
